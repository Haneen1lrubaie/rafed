## الأدوار والصلاحيات

### الزائر
- يتصفح الصفحة التعريفية فقط
- يستطيع الانتقال لتسجيل الدخول أو إنشاء حساب

### المتدرب
- يُنشئ حسابه مباشرة بدون كود دعوة
- يرفع ملفات المبادرات (PDF أو PPTX)
- يحذف مبادراته التي لم يتم تقييمها بعد
- يتابع حالة مبادراته (معلّقة / مقبولة / مرفوضة)
- يشاهد سجل حضوره الشخصي
- **لا يسجّل حضوره بنفسه** — الحضور يسجّله المشرف فقط

### المشرف
- يُنشئ حسابه بكود دعوة من الأدمن
- يُعيَّن على مجموعة واحدة أو أكثر
- يسجّل حضور وغياب المتدربين في مجموعته
- يراجع ويقيّم المبادرات (قبول / رفض مع ملاحظة)
- يصدّر سجل الحضور بصيغة Excel أو PDF
- يشاهد سجل الحضور الكامل لمجموعته

### الأدمن
- إنشاء وحذف المجموعات
- إضافة وإزالة مشرفين من المجموعات (مشرف واحد أو أكثر لكل مجموعة)
- إنشاء كودات دعوة للمشرفين فقط
- نقل المتدربين بين المجموعات
- تفعيل وتعطيل وحذف أي حساب
- مراجعة وتقييم مبادرات جميع المجموعات
- تصدير تقارير الحضور لأي مجموعة أو جميعها

| الطبقة | التقنية |
|--------|---------|
| Backend | Python + Flask |
